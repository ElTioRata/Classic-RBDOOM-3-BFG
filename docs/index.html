Classic RBDoom 3 BFG Edition

Welcome to Classic RBdoom 3 BFG's site

Classic RBDoom's goal is to restore various cut content from the Doom 3 BFG Edition, and add mod support for all 3 games that are included in it.

The site's goal is to gather all the available information about this mod in one place. This include update change logs, tutorials, demonstration videos and additional information.
